# Shared Utilities

This directory contains utility functions shared across the application.

## Utilities

### Encryption

- **crypto.ts** - AES-256-GCM encryption/decryption
- **keyDerivation.ts** - PBKDF2 key derivation
- **recoveryPhrase.ts** - BIP39 recovery phrase generation

### Database

- **migrations.ts** - Database migration system
- **queries.ts** - Common SQL queries

### Audio

- **audioProcessing.ts** - Audio format conversion
- **resampling.ts** - Sample rate conversion

### Validation

- **validators.ts** - Input validation functions
- **sanitizers.ts** - Data sanitization

### Logging

- **logger.ts** - Structured logging utility
