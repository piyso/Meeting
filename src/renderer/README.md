# Renderer Process

This directory contains the React UI code.

## Structure

- **components/** - React components
- **hooks/** - Custom React hooks
- **contexts/** - React contexts for state management
- **styles/** - CSS and styling

## Responsibilities

- User interface (React + Tiptap editor)
- Meeting list and navigation
- Transcript display with auto-scroll
- Note editor with Ctrl+Enter expansion
- Search interface
- Settings and preferences
