After downloading — three easy steps:

Because BlueArkive is free and independent, Apple shows a security warning. This quick unlock takes 30 seconds.

1

Drag to Applications

Open the .dmg file and drag BlueArkive into Applications.

2

Open Terminal

Press ⌘ Space, type Terminal, press Enter.

3

Paste & press Enter

Paste the command below and type your Mac password when asked.

Terminal
$ sudo xattr -cr /Applications/BlueArkive.app

That's it! Double-click BlueArkive from Applications — it opens perfectly, forever.

Installation is instant:

Run the installer and you're done. No terminal commands needed.

1

Run the Installer

Double-click BlueArkive-Setup.exe and follow the prompts.

2

Launch & Go

BlueArkive starts automatically. Find it in your Start Menu or Desktop.

SmartScreen warning? Click "More info" → "Run anyway". This appears because we're independent — not part of Microsoft's code signing program.

Quick start:

Download the AppImage and make it executable — done.

Terminal
$ chmod +x BlueArkive-*.AppImage && ./BlueArkive-*.AppImage

Or install the .deb/.rpm package for system integration.
