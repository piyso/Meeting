# Stereo Mix Quick Reference Card

**Print this page or keep it handy for quick reference when setting up audio capture.**

---

## Windows 11 - Quick Steps

1. Right-click speaker icon (🔊) → "Sound settings"
2. Scroll down → "More sound settings"
3. Click "Recording" tab
4. Right-click empty space → Check "Show Disabled Devices"
5. Right-click "Stereo Mix" → "Enable"
6. Right-click "Stereo Mix" → "Set as Default Device"
7. Play audio → Watch green meter move ✓

---

## Windows 10 - Quick Steps

1. Right-click speaker icon (🔊) → "Sounds"
2. Click "Recording" tab
3. Right-click empty space → Check "Show Disabled Devices"
4. Right-click "Stereo Mix" → "Enable"
5. Right-click "Stereo Mix" → "Set as Default Device"
6. Play audio → Watch green meter move ✓

---

## Keyboard Shortcut

**Windows + R** → Type `mmsys.cpl` → Press Enter  
_(Opens Sound control panel directly)_

---

## Common Issues & Quick Fixes

| Problem                 | Quick Fix                                                            |
| ----------------------- | -------------------------------------------------------------------- |
| Stereo Mix not in list  | Update audio driver or use VB-Audio Cable                            |
| No audio captured       | Set Stereo Mix as default device                                     |
| Audio too quiet         | Properties → Levels → Set to 100%                                    |
| Captures microphone too | Microphone Properties → Listen tab → Uncheck "Listen to this device" |

## Alternative Solutions

### Can't Enable Stereo Mix?

**Option 1: Use Microphone**

- Captures your voice only
- No setup required
- Works on all systems

**Option 2: Use Cloud Transcription**

- High accuracy
- Requires internet
- Free tier: 10 hours/month

**Option 3: VB-Audio Virtual Cable**

- Free software alternative
- Download: https://vb-audio.com/Cable/
- Creates virtual audio device

- [ ] Green meter moves when audio plays
